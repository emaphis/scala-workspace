# Scala notes

## Creating a new SBT project for Scala 3

> "navigate to project directory"

> sbt new scala/scala-seed.g8

> "Change into the project directory"  cd <your-project-name>

> "Edit the `build.sbt` file to configure the project setttings. "name" "version"

> sbt compile

> sbt run

> sbt test

> sbt console
